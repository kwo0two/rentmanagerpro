# **App Name**: Rent Manager Pro

## Core Features:

- Lease Agreement Input: 임대차 계약서에서 이름, 연락처, 호수, 임대 시작일 및 종료일, 임대료 등의 세부 정보를 캡처합니다.
- Automated Rent Roll: 갱신 날짜 및 미지불 잔액이 포함된 임대료 롤을 자동으로 생성합니다.
- Monthly Rent Calculation: 임대 조건에 따라 월 임대료를 자동으로 계산하고 업데이트합니다. LLM 도구를 사용하여 문서를 분석하고 확인합니다.
- Payment Tracking: 테넌트 지불 내역을 기록하고 호수별 지불 및 평균 지불 소요 일수와 같은 기타 분석을 표시합니다.
- Balance Calculation: 계정 잔액을 계산하고, 지불액과 미지불액을 표시하고, 연체 시 알립니다.

## Style Guidelines:

- 기본 색상: 신뢰성과 안정성을 전달하는 하늘색(#87CEEB).
- 배경 색상: 명확성을 보장하고 데이터에 집중할 수 있도록 밝은 회색(#F0F8FF), 거의 흰색.
- 액센트 색상: 대화형 요소 및 중요한 알림에 대한 청록색(#008080).
- 본문 및 제목 글꼴: 현대적이고 객관적인 모양을 홍보하기 위해 그로테스크 스타일의 산세리프체인 'Inter'.
- 다양한 기능을 나타내는 깨끗하고 전문적인 아이콘을 사용합니다.
- 깨끗하고 직관적인 레이아웃은 사용성을 향상시킵니다.
- 미묘한 전환을 사용하여 변경 사항 및 업데이트를 나타냅니다.
